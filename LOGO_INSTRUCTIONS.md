# تعليمات إضافة الشعار

## أين تضع ملف الشعار:

ضع ملف الشعار (logo.png) في **المجلد الرئيسي للمشروع** بجانب ملف `web_app.py`

```
camel_tools-master/
├── web_app.py          ← ملف التطبيق
├── logo.png            ← ضع الشعار هنا ✨
├── setup.py
├── README.rst
└── ...
```

## متطلبات الصورة:

- **اسم الملف**: `logo.png` (يجب أن يكون الاسم بالضبط)
- **الصيغة**: PNG (أو JPG/JPEG)
- **الحجم الموصى به**: 
  - العرض: 800-1200 بكسل
  - الارتفاع: 200-400 بكسل
  - أو نسبة 3:1 (عرض:ارتفاع)

## ملاحظات:

- إذا لم تجد الصورة، سيظهر العنوان النصي بدلاً منها
- الصورة ستظهر تلقائياً في:
  1. أعلى الصفحة الرئيسية
  2. في الصفحة الرئيسية نفسها

## مثال:

إذا كان اسم ملف الشعار مختلفاً، يمكنك تغييره في ملف `web_app.py` في السطر:
```python
logo_path = "logo.png"  # غيّر هنا إلى اسم ملفك
```
